# UTS Lab PBO 

Ini adalah project UTS untuk mata kuliah **Praktikum Pemrograman Berorientasi Objek (PBO)**.  
Project ini dibangun menggunakan **Java**, **JavaFX**, dan menggunakan file **FXML** untuk tampilan antarmuka.
